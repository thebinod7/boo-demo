## Run Locally

Setp 1: Clone the project

```bash
  https://github.com/thebinod7/boo-demo.git
```

Step 2: Go to the project directory and install dependencies

```bash
  cd boo-demo
  npm install
```

Step 3: Run project

```bash
   npm run dev
```

Step 4: API server is running at: http://localhost:3000

Step 5: Run test cases

```bash
   npm run test
```
